#+TITLE: Disjoint-Sets by Union-Find
#+AUTHOR: Gabriel Guillermo Saavedra Moreira

* NAME

=dsetuf= - disjoint-set (union-find) module

* LIBRARY

IRIS C Library (=ilib_c=)

* DESCRIPTION

The =dsetuf= module implements a disjoint-set (also called union-find) data structure over =set_size_t= integer elements in the rage =[0, size)=. Each element begins in its own singleton set; sets can be merged with =dset_join()=, and the representative (root) of the set containing an element can be queried with =dset_find()=.

Internally, each set is represented as a tree encoded in the parent array, where ~parent[x] = x~ identifies =x= as a root. Two optimizations keep operations fast:

- *Path compression* :: every call to =dset_find()= rewrites the parent pointers of all nodes visited so they point directly at the root, flattening the tree for future lookups.
- *Union by size* :: =dset_join()= always attaches the smaller tree's root under the larger tree's root, using the =size= array to track tree sizes, which bounds tree height logarithmically.

Together these give amortized near-constant time per operation.

The =disjoint_set_t= structure is defined as:

#+BEGIN_SRC c
typedef struct {
        set_size_t  set_count;
        set_size_t *size;
        int        *parent;
} disjoint_set_t;
#+END_SRC

=set_size_t= is an alias for unsigned int.

Both =size= and =parent= are dynamically allocated arrays, using the allocator specified by =DSUF_MALLOC=; see [[*IMPLEMENTATION MACROS][IMPLEMENTATION MACROS]].

** IMPLEMENTATION MACROS

The following preprocessor macros control code generation and are checked at include time:

- *DSUF_IMPLEMENTATION* :: Must be defined in exactly one translation unit before including ~<dsetuf.h>~ to emit function bodies. All other translation units should include the header without defining it, receiving declarations only.

- *DSUF_DECL* :: Overrides the storage/linkage specifier applied to every function. Defaults to ~static inline~ if left undefined.

- *DSUF_MALLOC*, *DSUF_FREE* :: Override the allocator used by =dset_construct()= and =dset_destroy()=. Both must be defined together, or neither; defining only one is a compile-time error. If neither is defined, both default to the standard library's =malloc()= and =free()=.

* NOTES

No bounds checking is performed on the element index =x= passed to =dset_find()=, =dset_join()=, =dset_joined()=, or =dset_cardinality()=. Passing a value outside =[0, size)= is undefined behavior.

=dset_find()= is implemented recursively. Path compression keeps subsequent calls shallow, but the first call on a long, uncompressed chain recurses to a depth proportional to the chain length.

=dsetuf= is not thread-safe; concurrent calls that mutate the same =disjoint_set_t= from multiple threads (including concurrent =dset_find()= calls, since path compression writes to =parent=) are a data race.

=dset_joined()= and =dset_cardinality()= take a ~const disjoint_set_t *~ but internally discard the =const= qualifier to call =dset_find()=, since path compression mutates =parent= even though the operation is logically a read. This is intentional (logical constness), not an oversight.

* EXAMPLES

The following program builds a minimum spanning tree over a small weighted graph using Kruskal's algorithm, using =dsetuf= to detect and reject edges that would form a cycle.

*** Kruskal program
#+BEGIN_SRC c
#define DSUF_IMPLEMENTATION
#include "dsetuf.h"

#include <stdio.h>
#include <stdlib.h>

typedef struct {
	int u, v, weight;
} edge_t;

int cmp_edge(const void *a, const void *b)
{
	return ((const edge_t *)a)->weight
	     - ((const edge_t *)b)->weight;
}

int main(void)
{
	edge_t edges[] = {
		{0, 1, 4}, {0, 2, 1}, {1, 2, 2},
		{1, 3, 5}, {2, 3, 8},
	};
	int n_edges = sizeof(edges) / sizeof(edges[0]);
	int n_vertices = 4;

	qsort(edges, n_edges, sizeof(edge_t), cmp_edge);

	disjoint_set_t ds;
	dset_construct(&ds, n_vertices);

	int total_weight = 0;

	for (int i = 0; i < n_edges; ++i) {
		edge_t e = edges[i];

		if (dset_joined(&ds, e.u, e.v))
			continue; /* would form a cycle */

		dset_join(&ds, e.u, e.v);
		total_weight += e.weight;
		printf("added edge %d-%d (weight %d)\n",
		       e.u, e.v, e.weight);
	}

	printf("total MST weight: %d\n", total_weight);

	dset_destroy(&ds);
	return 0;
}
#+END_SRC
